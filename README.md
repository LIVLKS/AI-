# AI作业批改系统（教师端 + 学生端）

基于 Python + Gradio 的简易智能作业批改系统，支持教师布置填空题、学生在线作答、AI 自动评分并按可信度分组，教师复核后可发布评语与最终成绩。

## 环境依赖
- Python 3.8+
- Gradio
- 
## 功能概览

### 教师端（teacher_app.py，本地端口 7860）
- **作业工作台**：查看各班级作业批改进度（AI已批 / 待复核 / 已发布）
- **布置作业**：按模板格式添加填空题，自动发布到指定班级
- **AI批改面板**：
  - 按可信度分组展示学生提交（高 / 中 / 低）
  - 高可信组：一键全部发布
  - 中可信组：随机抽查辅助决策
  - 低可信组：逐份人工复核，可修改分数、添加评语
- **评语助手**：AI 生成个性化评语草稿，教师可修改后发布
- **配置中心**：动态调整高/中可信度阈值

### 学生端（`student_app.py`，端口 7861）
- **我的作业**：查看本班作业，在线填空作答
- **即时AI批改**：提交后立刻看到每题得分、对错及可信度
- **错题详情**：标记错题（错因定位、知识解析暂为“暂无”）
- **温习订正**：教师发布后，可重新作答进行二次批改
